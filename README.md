# ServiceNow End-to-End Business Process (ITSM + CSM + HRSD + ITAM)

Flagship project describing a single complex process that uses multiple ServiceNow products together (hire-to-retire & customer support).

## Overview

This repo focuses on architecture and documentation for an enterprise scenario where HR, IT, customer support, operations, and asset management all live on one platform, with automated hand-offs.

## Scenario Highlights

- New hire onboarding (HRSD + ITAM + ITSM).
- Employee supporting customers via CSM.
- Infrastructure monitored and auto-healed via ITOM integrated with ITSM.
- Assets tracked and refreshed through ITAM.
- Exit process with offboarding, asset recovery, and access revocation.

## Files in this repo

- `/docs/e2e-process-flow.md` – step-by-step journey from pre-hire to exit.  
- `/docs/module-integration.md` – how data and workflows cross modules.  
- `/docs/use-cases.md` – short stories you can use in interviews.

This project is ideal for talking about solution architecture and cross-module design.
